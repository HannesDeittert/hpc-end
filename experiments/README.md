# Experiments

Kleine, reproduzierbare Experimente und Skripte, die nicht Teil der Library/UI sind.

Empfohlen:
- Jedes Experiment in eigenem Unterordner:
  - `config/` (YAML/JSON)
  - `run.py`
  - `README.md` mit Ziel + Kommando
- Outputs nach `results/<experiment>/...` und Logs nach `logs/<experiment>/...`.
